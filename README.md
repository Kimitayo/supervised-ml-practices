# Supervised Learning Coursework

This repository contains the practical assignments, experiments, and final project developed during the **Supervised Machine Learning** course.

Although the course covers a range of supervised learning techniques, its main focus is on **dataset preprocessing and cleaning**, including data transformation, sampling, dimensionality reduction, feature extraction, and model evaluation. The final project applies these concepts to an image classification task involving **dog and cat breed separation**.

## Course Overview

The course was designed to build practical skills in supervised machine learning through:

- Lectures supported by slides, whiteboard explanations, and presentations
- Discussions of concepts and problem-solving strategies
- Hands-on laboratory activities

Assessment was based on practical assignments during the course and a final project integrating the concepts learned throughout the semester.

## Main Topics Covered

### 1. Data Preprocessing and Cleaning
- Data preprocessing concepts
- Dataset cleaning
- Data transformation
- Data reduction through sampling
- Dimensionality reduction
- PCA
- Feature selection

### 2. Image Processing and Feature Extraction
- HOG (Histogram of Oriented Gradients)
- LBP (Local Binary Patterns)
- Image preprocessing practice

### 3. Training, Testing, and Validation
- Train/test strategies
- Cross-validation
- Statistical testing

### 4. Supervised Learning Algorithms
- k-Nearest Neighbors (k-NN)
- Decision Trees
- Naive Bayes
- Neural Networks
- Deep Learning
- Classifier ensembles
- Transfer Learning

### 5. Final Project
- Development of a supervised learning project for **dog and cat breed classification**
- Model selection and evaluation
- Application of preprocessing, feature extraction, and classification techniques
